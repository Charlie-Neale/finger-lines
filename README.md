# finger-lines

Real-time browser-based hand tracking visual effects, powered by MediaPipe Hands.

## Run

```bash
npm install
npm run dev
```

Then open the URL Vite prints (usually `http://localhost:5173`) and allow camera access.

## Controls

| Key | Effect |
|-----|--------|
| `1` | Invert |
| `2` | Grayscale |
| `3` | Pixelate |
| `4` | Glitch |
| `5` | Hue rotate |

See `CLAUDE.md` for the full spec.
